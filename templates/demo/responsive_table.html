How it Works

Desktop View (> 768px)

  • Standard HTML <table> structure
  • thead is visible
  • Rows display horizontally
  • Zebra striping and hover effects active

Mobile View (< 768px)

  • thead is hidden via CSS
  • Each tr becomes a card (block display, shadow, margin)
  • Each td becomes a flex container
  • ::before pseudo-element displays the column header using attr(data-label)

Key CSS Logic

@media (max-width: 768px) {
  /* Reset table layout */
  table, thead, tbody, th, td, tr {
    display: block;
    width: 100%;
  }

  /* Hide headers visually */
  thead tr {
    position: absolute;
    top: -9999px;
    left: -9999px;
  }

  /* Card style for rows */
  tr {
    margin-bottom: 1.5rem;
    border: 1px solid #e2e8f0;
    border-radius: 0.75rem;
    background-color: #ffffff;
    box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1);
  }

  /* Cell style */
  td {
    border: none;
    border-bottom: 1px solid #f1f5f9;
    position: relative;
    padding: 0.75rem 1rem;
    display: flex;
    justify-content: space-between;
    align-items: center;
    text-align: right;
  }

  /* Label style via pseudo-element */
  td::before {
    content: attr(data-label);
    font-weight: 600;
    color: #64748b;
    text-transform: uppercase;
    font-size: 0.75rem;
    margin-right: auto;
    text-align: left;
  }
}
